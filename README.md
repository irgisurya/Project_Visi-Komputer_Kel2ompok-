# Project_Visi-Komputer_Kelompok-2
**Python packages required:**
* cvzone 1.5.6
* mediapipe 0.8.9.1
* pyfirmata 1.1.0
* pyglet 1.5.23

**Arduino Configuration**
- Open IDE Arduino
- Select File -> Example -> Firmata -> StandardFirmata
- Select Tools -> Board -> Arduino/Genuino Uno
- Select Tools -> Port -> *choose your port COM*
- Upload
